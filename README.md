🐍 Exercícios de Python: Mundo 1

Curso em Vídeo - Prof. Gustavo Guanabara

Este repositório tem as minhas resoluções para os desafios do Mundo 1 de Python. O objetivo foi aprender a base da linguagem, desde a instalação até condições simples.

🚀 O que aprendi neste módulo:

Tratamento de Dados: Tipos como int, float, bool e str.

Operações Aritméticas: Contas matemáticas e a biblioteca math.

Módulos: Como importar ferramentas prontas para o código.

Manipulação de Texto: Cortar e analisar frases ou palavras.

Estruturas de Controlo: Uso de if e else para tomar decisões.

Cores no Terminal: Como deixar o texto colorido com códigos ANSI.

📁 Estrutura do Repositório

Ficheiros do ex001.py ao ex035.py com os desafios resolvidos e algumas notas.
